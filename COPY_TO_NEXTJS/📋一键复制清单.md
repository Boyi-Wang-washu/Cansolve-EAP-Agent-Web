## ⚠️ 重要提醒

### Flask 在哪里？
```
Flask 不在 Next.js 项目里！
Flask 在 memorized/ 文件夹独立运行！

memorized/          Next.js 项目/
├── app.py          ├── src/
├── (Flask 代码)     │   ├── lib/chatApi.ts ← 复制
                    │   └── components/... ← 复制
```

### 两个服务同时运行
```
终端 1: cd memorized && python app.py  (5000端口)
终端 2: cd nextjs && npm run dev       (3000端口)
```
